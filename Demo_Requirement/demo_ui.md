# 核心看板模块 (Dashboard)-

- 实时负荷表盘： 实时显示当前时刻的 load_kw（暂定手动输入）。通过颜色（绿、黄、红）直观反映负荷是否接近变压器容量。
- 实时功率流向图 (Sankey Diagram)： 动态显示电力的流向（光伏-充电桩、光伏 -储能、电网-充电桩等）。能直观展示绿电自给率。
- 关键指标对比：
  -  今日预测峰值： 预测模型算出的今日最高负荷及其发生时间。
  -  预测偏差 (MAPE)： 实时计算过去一小时内预测值与实际值的误差。
  -  当前电价区间： 显示当前是峰、平还是谷时段（联动price电价特征）。
  -  实时净负荷 (Net Load)： Pnet=Pload−Ppv。若为负值，说明光伏有余量，可充入储能或卖给电网。
  -  绿电替代率： 今日充电总量中，光伏发电所占的比例



# 负荷预测曲线对比模块 (Visualization)

使用交互式图表（如 ECharts 或 Plotly）。

- 多线同屏对比：
  - 实测曲线（实线）： 历史至今的 15min 实际负荷。
  - 预测曲线（虚线）： 模型输出的未来 24 小时预测值。
  - 置信区间（阴影层）：引入贝叶斯，可显示预测的上下界波动范围（Uncertainty Band）。
- 历史同期对比： 自动拉取昨日（lag_96）或上周同期的曲线，辅助判断今日负荷是否异常。



# 气象与外部环境模块 (Contextual Metadata)

光伏预测对天气极度敏感，前端必须引入气象看板。

- 辐照度与云量监测： 实时显示当前的太阳辐射强度（W/m2）和云层覆盖率。
- 天气预警提示： 接入气象 API，若预测到未来 2 小时有强降雨，前端应弹出提醒：光伏出力预计骤降，请调整储能策略以应对充电负荷。



# 特征敏感度分析模块 (Feature Inspection)（电动汽车负荷预测部分）

作为内部工具，帮助算法人员分析模型为什么给出了这个预测结果。

- 影响因素热力图： 展示当前预测受哪些特征影响最大（例如：电价波动对负荷的贡献度）。
- 输入变量实时监测： 实时显示模型输入的 price、is_weekend、rolling_mean_4 等参数的变化情况。



# 调度建议与控制策略、预警与报警配置模块 (Alerting)

将预测能力转化为实际的运维生产力。

- 光伏利用建议：
  - 预计 12:00-14:00 光伏盈余 50kW，建议开启有序充电或储能充电。
  - 预计傍晚负荷高峰期光伏为 0，建议目前保留储能电量至 18:00 释放。
- 收益分析看板： 实时计算通过“光伏直供”节省了多少电费（对比从电网购电的价格）。
- 阈值预警： 当预测到未来 1 小时内负荷将超过设定阈值（如变压器 80% 额定功率）时，系统自动推送弹窗或企业微信/钉钉消息。
- 数据缺失报警： 如果 TCN 输入所需的 15min 数据流中断，前端应显著提示“数据链路异常”，防止模型输出错误结果。



# 策略模拟与控制建议模块 (Decision Support)

这是光储充项目的升华，将预测结果导向控制。
- 储能充放电建议： 根据预测的负荷曲线，自动给出最优储能调度建议。例如：“预测 14:00 进入电价高峰且负荷激增，建议 12:00 开始为储能充电”。
- 充电桩限流建议： 在预测到电力过载时，建议后端调整充电桩的最大输出功率。
