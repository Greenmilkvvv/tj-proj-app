# 业务理解

- 软件名称：光储充智能预测系统
- 产品定位：面向光伏与充电站运维场景的智能预测工具，基于深度学习模型预测光伏发电功率和充电负荷，生成储能充放电调度策略建议，辅助能源调度决策。
- 面向领域 / 行业：新能源/能源管理
- 核心价值：通过 AI 预测光伏出力和充电负荷，计算绿电消纳比例，提供储能充放电策略建议，提升新能源利用效率，降低用电成本。
- 证据文件：`业务理解证据.md`

## 目标用户

- 光伏电站运维人员
- 充电站运营管理人员
- 能源调度决策者

## 主要业务功能

- 综合预测
- 气象查询
- 数据探索
- 误差分析
- 模型信息

## 功能说明

- 综合预测：用户设置模型类型（光伏/充电/综合）、预测步数（15分钟/1小时/24小时）、电价和当前负荷参数，系统调用深度学习模型进行多步迭代预测，生成光伏出力曲线、充电负荷曲线、绿电消纳比例、光伏峰值及充电峰值等指标，并输出分时段储能充放电调度策略建议。
- 气象查询：实时获取指定位置（宁波坐标）的气象数据，包括当前温度、辐照度、云量、降雨量及天气预警信息，并以表格和辐照度趋势图形式展示未来数小时的气象预报。
- 数据探索：提供历史数据的多维可视化分析，包括数据集概览（统计摘要）、单日负荷曲线查询、多日聚合负荷曲线（均值±标准差）、历史日负荷曲线比较、特征相关性分析、以及24小时分时负荷画像。
- 误差分析：通过回测评估模型精度，包含充电负荷回测（实际值vs预测值对比）、光伏功率回测、充电误差分布直方图、光伏误差分布直方图、以及按小时统计的误差趋势，帮助用户了解模型在各时段的预测表现。
- 模型信息：展示光伏预测模型（LSTM+GAN）和充电预测模型（TCN-Attention-LSTM）的架构说明、参数配置、权重文件位置等技术信息。

4. 切换至数据探索标签页查看历史数据图表
5. 切换至误差分析标签页评估模型精度
6. 切换至模型信息标签页了解技术细节

## 操作手册页面/流程模块

1. 综合预测：打开软件首页即显示综合预测标签页
   - 使用场景：运维人员需要了解未来数小时的光伏出力和充电负荷，以便提前安排储能充放电计划
   - 操作要点：在模型选择下拉框中选择预测类型：光伏、充电或综合；在预测步数下拉框中选择预测时长：15分钟、1小时或24小时；根据需要修改电价和当前负荷参数；点击预测按钮发起预测请求
2. 气象查询：点击页面顶部的气象标签页
   - 使用场景：用户需要查看当前气象条件和未来气象变化趋势，辅助判断光伏出力的天气因素
   - 操作要点：点击气象标签页进入气象查询界面；查看页面顶部当前气象摘要信息；浏览气象预报表格了解未来天气趋势；查看辐照度趋势图了解辐照度变化
3. 数据探索：点击页面顶部的数据探索标签页
   - 使用场景：用户需要了解历史数据分布、负荷规律、特征相关性等，为运营决策提供数据支撑
   - 操作要点：点击数据探索标签页进入界面；查看数据集概览统计表了解数据概况；在日期输入框中输入日期（如2026-01-02），查询该日负荷曲线；选择起始日期和结束日期后查看聚合负荷曲线（含日均值和标准差范围）
4. 误差分析：点击页面顶部的误差分析标签页
   - 使用场景：用户需要了解模型预测精度，评估模型在不同时段的预测表现，判断预测结果的可靠性
   - 操作要点：点击误差分析标签页进入界面；查看充电负荷回测图表和汇总指标，了解充电模型精度；查看光伏功率回测图表和汇总指标，了解光伏模型精度；查看充电误差分布直方图，了解误差集中在哪个区间
5. 模型信息：点击页面顶部的模型信息标签页
   - 使用场景：技术人员需要了解模型的架构设计、参数配置和权重文件位置
   - 操作要点：点击模型信息标签页进入界面；阅读光伏预测模型的架构和参数说明；阅读充电预测模型的架构和参数说明
