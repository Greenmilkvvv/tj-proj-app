## 二、说明

光储充智能预测系统 V1.0适用于新能源/能源管理场景。用户进入系统后，可以围绕实际工作内容完成账号进入、业务创建、过程查看、结果确认和资料管理等操作。

日常使用时，光伏电站运维人员、充电站运营管理人员、能源调度决策者可以按照页面提示从入口进入相应页面，查看当前业务状态，并根据页面中的按钮、输入框、列表或弹窗继续处理。通过 AI 预测光伏出力和充电负荷，计算绿电消纳比例，提供储能充放电策略建议，提升新能源利用效率，降低用电成本。

面向光伏与充电站运维场景的智能预测工具，基于深度学习智能服务预测光伏发电功率和充电负荷，生成储能充放电调度策略建议，辅助能源调度决策。

本软件是一款面向光储充一体化站点的智能预测工具，通过深度学习智能服务对光伏发电功率和充电负荷进行预测，计算绿电消纳比例，生成储能充放电调度策略建议。软件适用于光伏电站、充电站等新能源站点的运维管理和能源调度决策场景。

联合预测功能：支持光伏发电功率预测、充电负荷预测和联合预测三种模式。用户可自定义预测步数（15分钟、1小时或24小时）及电价参数，系统通过深度学习智能服务进行多步迭代预测，输出光伏出力曲线和充电负荷曲线，计算绿电消纳比例。

气象数据功能：自动获取站点所在区域的实时气象数据，展示当前温度、辐照度、云量、降雨量等信息，并识别极端天气预警。同时以表格和趋势图形式展示未来数小时的气象预报和辐照度变化趋势，帮助用户判断光伏出力的天气因素。

本手册用于说明软件的用途、功能特点、运行要求和页面操作流程。各功能章节按用户能够看到的页面、入口、按钮、输入项、提示信息和处理结果进行说明。

## 三、功能特点

用户可在联合预测页面对光伏发电功率和充电负荷进行预测，计算绿电消纳比例。页面上的智能服务选择下拉框（光伏、充电、综合）、预测步数下拉框（15分钟单步预测、1小时预测、24小时预测）、电价输入框、负荷输入框和预测按钮会集中呈现当前可操作内容，用户处理完成后可以看到预测完成后图表自动刷新，展示预测曲线和汇总指标面板显示总光伏出力、总充电负荷、绿电消纳比例等数据。

在气象数据页面中，用户主要处理获取站点所在区域的实时气象数据和预报信息。系统把当前时间显示、当前温度数值、辐照度数值、云量百分比和降雨量数值放在当前操作区域，处理结束后会反馈页面自动加载气象数据，显示当前温度、辐照度、云量、降雨量、如存在极端天气预警，显示红色警告提示和辐照度趋势图以折线图展示未来辐照度变化。

数据探索页面关注的是对历史充电和光伏数据进行可视化探索。用户通过数据集概览统计表、日期输入框（查询单日负荷）、日期范围选择器（起始日期、结束日期）、单日负荷曲线图表和聚合负荷曲线图表确认当前状态，并在操作结束后获得各图表在查询或选择日期后自动刷新渲染、无数据时图表显示提示信息和聚合负荷曲线显示均值线和±1σ标准差带。

用户可在误差分析页面评估预测智能服务的准确性，通过回测和误差分布分析展示智能服务精度。页面上的充电负荷回测图表：实际值曲线与预测值曲线对比、充电负荷回测汇总指标（MAE、RMSE等）、光伏功率回测图表：实际值曲线与预测值曲线对比、光伏功率回测汇总指标（MAE、RMSE等）和充电误差分布直方图会集中呈现当前可操作内容，用户处理完成后可以看到回测图表以双曲线对比实际值和预测值、汇总指标面板显示 MAE 和 RMSE 等精度指标和误差分布直方图展示误差的频次分布。

在策略建议页面中，用户主要处理根据联合预测结果生成分时段储能充放电调度策略建议。系统把生成策略建议按钮、提示文字：请先在联合预测 Tab 执行预测，再点击按钮生成策略建议、策略建议表格：时间、光伏出力(kW)、充电负荷(kW)、净差值(kW)、调度动作和未生成策略时的空状态提示放在当前操作区域，处理结束后会反馈点击生成策略建议按钮后，表格自动填充各时段策略数据、策略表格每行显示对应时段的调度动作和盈余时标记为储能充电，缺口时标记为储能放电，平衡时标记为待机。

## 五、联合预测

用户可在联合预测页面对光伏发电功率和充电负荷进行预测，计算绿电消纳比例。用户打开软件首页即显示联合预测标签页。

运维人员需要了解未来数小时的光伏出力和充电负荷，以便提前安排储能充放电计划。

用户在联合预测页面会看到智能服务选择下拉框（光伏、充电、综合）、预测步数下拉框（15分钟单步预测、1小时预测、24小时预测）、电价输入框、负荷输入框、预测按钮、光伏出力曲线图表、充电负荷曲线图表和汇总指标面板：总光伏出力、总充电负荷、绿电消纳比例、光伏峰值及时间、充电峰值等信息，并可依据页面显示继续处理。

使用该功能时，用户先在智能服务选择下拉框中选择预测模式：光伏、充电或综合，随后在预测步数下拉框中选择预测时长：15分钟、1小时或24小时，接着根据需要修改电价和当前负荷参数，之后点击预测按钮发起预测请求，再查看上方光伏出力曲线和充电负荷曲线，最后查看右侧汇总指标面板的关键数据。

页面会按照预测步数限定为15分钟、1小时、24小时三个选项、电价必须为正数和负荷必须为非负数等规则限制或提示用户。处理结束后，用户可以看到预测完成后图表自动刷新，展示预测曲线和汇总指标面板显示总光伏出力、总充电负荷、绿电消纳比例等数据。

## 六、气象数据

用户可在气象数据页面获取站点所在区域的实时气象数据和预报信息。用户点击页面顶部的气象数据标签页。

用户需要查看当前气象条件和未来气象变化趋势，辅助判断光伏出力的天气因素。

该部分提供当前时间显示、当前温度数值、辐照度数值、云量百分比、降雨量数值、天气预警提示信息、气象预报表格：时间、辐照度、云量、降雨量、天气描述和辐照度趋势折线图等页面内容，用户可据此查看状态、填写信息或选择下一步操作。

在该页面中，用户先点击气象数据标签页进入气象查询界面，随后查看页面顶部当前气象摘要信息，接着浏览气象预报表格了解未来天气趋势，最后查看辐照度趋势图了解辐照度变化。

页面反馈通常包括页面自动加载气象数据，显示当前温度、辐照度、云量、降雨量、如存在极端天气预警，显示红色警告提示和辐照度趋势图以折线图展示未来辐照度变化。

## 七、数据探索

用户可在数据探索页面对历史充电和光伏数据进行可视化探索。用户点击页面顶部的数据探索标签页。

用户需要了解历史数据分布、负荷规律、特征相关性等，为运营决策提供数据支撑。

页面上主要呈现数据集概览统计表、日期输入框（查询单日负荷）、日期范围选择器（起始日期、结束日期）、单日负荷曲线图表、聚合负荷曲线图表、日负荷曲线对比图表、相关性分析条形图和24小时负荷画像图等内容，这些内容用于帮助用户确认当前位置和可执行操作。

实际操作时，用户先点击数据探索标签页进入界面，随后查看数据集概览统计表了解数据概况，接着在日期输入框中输入日期（如2026-01-02），查询该日负荷曲线，之后选择起始日期和结束日期后查看聚合负荷曲线（含日均值和标准差范围），再选择起始日期和结束日期后查看历史日负荷曲线叠加对比，继续查看特征相关性分析条形图，了解各特征与负荷的关联程度，最后查看24小时分时负荷画像图，了解各时段负荷分布。

操作过程中需要注意日期格式必须为YYYY-MM-DD、日期范围最多60天，超出时自动截断和起始日期不能晚于结束日期。操作完成后，系统会显示各图表在查询或选择日期后自动刷新渲染、无数据时图表显示提示信息和聚合负荷曲线显示均值线和±1σ标准差带。

## 八、误差分析

用户需要了解智能服务预测精度，评估智能服务在不同时段的预测表现，判断预测结果的可靠性。

用户在误差分析页面会看到充电负荷回测图表：实际值曲线与预测值曲线对比、充电负荷回测汇总指标（MAE、RMSE等）、光伏功率回测图表：实际值曲线与预测值曲线对比、光伏功率回测汇总指标（MAE、RMSE等）、充电误差分布直方图、光伏误差分布直方图、按小时充电误差趋势图和按小时光伏误差趋势图等信息，并可依据页面显示继续处理。

使用该功能时，用户先点击误差分析标签页进入界面，随后查看充电负荷回测图表和汇总指标，了解充电智能服务精度，接着查看光伏功率回测图表和汇总指标，了解光伏智能服务精度，之后查看充电误差分布直方图，了解误差集中在哪个区间，再查看光伏误差分布直方图，了解误差集中在哪个区间，最后查看按小时误差趋势图，了解智能服务在各时段的预测表现差异。

处理结束后，用户可以看到回测图表以双曲线对比实际值和预测值、汇总指标面板显示 MAE 和 RMSE 等精度指标、误差分布直方图展示误差的频次分布和按小时误差趋势图展示各时段的平均误差和波动范围。

## 九、策略建议

用户可在策略建议页面根据联合预测结果生成分时段储能充放电调度策略建议。用户点击页面顶部的策略建议标签页。

用户在联合预测标签页完成预测后，需要查看各时段的储能电池充放电调度建议。

该部分提供生成策略建议按钮、提示文字：请先在联合预测 Tab 执行预测，再点击按钮生成策略建议、策略建议表格：时间、光伏出力(kW)、充电负荷(kW)、净差值(kW)、调度动作和未生成策略时的空状态提示等页面内容，用户可据此查看状态、填写信息或选择下一步操作。

在该页面中，用户先先在联合预测标签页完成预测，随后点击策略建议标签页进入界面，接着点击生成策略建议按钮，之后查看生成的策略建议表格，了解各时段的光伏出力、充电负荷、净差值，最后根据调度动作列的建议（储能充电/储能放电/待机）安排储能电池充放电计划。

## 十一、常见问题解答

问题：预测结果不准确怎么办？
解决方法：可以切换至误差分析标签页查看智能服务在不同时段的精度表现，了解预测的可靠范围。预测结果受气象数据和负荷波动影响，建议结合实际情况综合判断。

问题：如何调整预测参数？
解决方法：在联合预测标签页中修改电价、当前负荷参数，在下拉框中选择不同的预测步数（15分钟、1小时或24小时）。参数修改后点击预测按钮即可获得更新结果。

问题：气象数据从哪里获取？
解决方法：气象数据通过 Open-Meteo 免费气象 API 实时获取，站点位置设定为宁波坐标。数据包括温度、辐照度、云量、降雨量等信息。

问题：系统支持哪些预测模式？
解决方法：支持光伏预测、充电预测和联合预测三种模式。光伏预测采用 LSTM 加 GAN 对抗训练智能服务，充电预测采用 TCN-Attention-LSTM 混合深度智能服务。两个智能服务可独立运行，也可联合运行。

## 十二、术语表

| 术语 | 解释 |
| --- | --- |
| 光伏出力 | 光伏发电系统在特定时刻输出的电功率，单位为千瓦（kW）。 |
| 充电负荷 | 充电站在特定时刻的用电功率需求，单位为千瓦（kW）。 |
| 绿电消纳比例 | 光伏发电量占充电用电总量的百分比，数值越高表示新能源利用效率越好。 |
| 储能调度策略 | 根据光伏出力和充电负荷的供需关系，系统给出的储能电池充电或放电行动建议。 |
| 回测 | 使用历史数据检验智能服务预测精度的方法，将智能服务对历史时段的预测值与实际观测值进行对比。 |
| 辐照度 | 太阳辐射照射到地面单位面积上的功率，单位为瓦每平方米（W/m²），是影响光伏出力的关键气象因素。 |
